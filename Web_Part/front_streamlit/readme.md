# Streamlit
Streamlit을 이용한 모델 prototype 

## Getting Started
0. Python requirements
    `Python` : 3.7

1. Installation
    1. 가상 환경을 설정합니다.
    
    2. 프로젝트의 의존성을 설치합니다.
        ```shell
        > pip install -r requirements.txt 
        ``` 
    3. 애플리케이션을 실행합니다
        ```shell
        > streamlit run app.py --server.port 30001
        ```
